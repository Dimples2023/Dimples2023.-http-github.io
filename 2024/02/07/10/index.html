1.最长回文子串

知识点：字符串 动态规划

描述：

给你一个字符串 s，找到 s 中最长的回文子串。

如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。

示例 1：

输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。

示例 2：

输入：s = "cbbd"
输出："bb"

题解：

1.暴力-超时
class Solution:
    def longestPalindrome(self, s: str) -> str:
        if len(s)==1:
            return 1
        max_len=1
        for i in range(len(s)):
            for j in range(i+1,len(s)):
                start=i
                end=j
                flag=0
                while(end>start):
                    if s[start]!=s[end]:
                        flag=1
                        break
                    start+=1
                    end-=1
                if flag==0:
                    if max_len<j-i+1:
                        max_len=j-i+1
                        m=i
                        n=j
        return s[m:n+1]
2.动态规划

思路：

1.构造二维矩阵dp，维度为len(s)*len(s)，dp[left] [right]表示子串s[left:right+1]是否为回文串，left和right均为下标索引，范围在0~len(s)之间，且left<right，因此dp矩阵中的每一个变量均为布尔量；
2.构造状态转移方程，考虑三种情况，
（1）left = right，当前子串只有一个字符，一定为回文串；
（2）right = left + 1，当前子串只有两个字符，判断这两个字符是否相等；
（3）right > left + 1，当前子串有多个字符，判断根据首位字符是否相等以及除去首位字符剩余子串是否为回文串判定当前子串。
状态转移方程为：

注意：在状态转移方程中，我们是从长度较短的字符串向长度较长的字符串进行转移的，因此一定要注意动态规划的循环顺序

class Solution:
    def longestPalindrome(self, s: str) -> str:
        #一个字符必为回文
        if len(s)==1:
            return s
        # dp[i][j] 表示 s[i..j] 是否是回文串
        dp=[[False]*len(s) for _ in range(len(s))]
        for i in range(len(s)):
            dp[i][i]=True
        #记录最大回文长度以及起始位置
        max_len,max_start=1,0
        
        # 递推开始
        # 先枚举子串长度
        for L in range(2,len(s)+1):
            # 枚举左边界
            for i in range(len(s)):
                # 由 L 和 i 可以确定右边界
                j=i+L-1
                # 如果右边界越界，就可以退出当前循环
                if j>=len(s):
                    break
                #第一种情况
                if j-i<=1:
                    dp[i][j]=(s[i]==s[j])
                #第二种情况
                else:
                    dp[i][j]=(s[i]==s[j]) and (dp[i+1][j-1])
                cur_len=j-i+1
                 # 只要 dp[i][j] == true 成立，就表示子串 s[i..j] 是回文，此时记录回文长度和起始位置
                if dp[i][j] and cur_len>max_len:
                    max_len=cur_len
                    max_start=i
        return s[max_start:max_start+max_len]
