1.出界的路径数

题目:给你一个大小为 m x n 的网格和一个球。球的起始坐标为 [startRow, startColumn] 。你可以将球移到在四个方向上相邻的单元格内（可以穿过网格边界到达网格之外）。你 最多 可以移动 maxMove 次球。

给你五个整数 m、n、maxMove、startRow 以及 startColumn ，找出并返回可以将球移出边界的路径数量。因为答案可能非常大，返回对 109 + 7 取余 后的结果。

示例 1：

输入：m = 2, n = 2, maxMove = 2, startRow = 0, startColumn = 0
输出：6

示例 2：

输入：m = 1, n = 3, maxMove = 3, startRow = 0, startColumn = 1
输出：12

题解：
思路：
动态规划。
但是这道题要反过来想，就是先在地图边缘补一圈1，因为每个出界点只有一种办法出来。然后我们倒推回去，看走到某个位置会触发到的出界数。

class Solution:
    def findPaths(self, m: int, n: int, N: int, i: int, j: int) -> int:
        # 定义dp[k][i][j]为用k步走到位置[i][j]处能出界的路径数
        dp = [[[0]*(n+2) for _ in range(m+2)] for _ in range(N+1)]
        mod = 1e9+7

        if N == 0:  return 0
        # 初始化边缘
        for k in range(N+1):
            for x in range(m+2):
                for y in range(n+2):
                    if x == 0 or y == 0 or x == m + 1 or y == n + 1:
                        dp[k][x][y] = 1
        # 开始遍历地图
        for k in range(1,N+1):
            for x in range(1,m+1):
                for y in range(1, n+1):
                    dp[k][x][y] = (dp[k][x][y] + dp[k-1][x-1][y] + \
                                dp[k-1][x+1][y] + dp[k-1][x][y-1] + \
                                dp[k-1][x][y+1]) % mod
        return dp[N][i+1][j+1]

2.有效的正方形

题目：给定2D空间中四个点的坐标 p1, p2, p3 和 p4，如果这四个点构成一个正方形，则返回 true 。

点的坐标 pi 表示为 [xi, yi] 。 输入没有任何顺序 。

一个 有效的正方形 有四条等边和四个等角(90度角)。

示例 1:

输入: p1 = [0,0], p2 = [1,1], p3 = [1,0], p4 = [0,1]
输出: True

示例 2:

输入：p1 = [0,0], p2 = [1,1], p3 = [1,0], p4 = [0,12]
输出：false

示例 3:

输入：p1 = [1,0], p2 = [-1,0], p3 = [0,1], p4 = [0,-1]
输出：true

题解：
思路：
判定正方形的方法：计算四个点的之间距离，如果是正方形，必然会有四条边相等，并且，它的两条对角线也相等。

class Solution:
    def validSquare(self, p1: List[int], p2: List[int], p3: List[int], p4: List[int]) -> bool:
        # 去掉四个点都重合的情况
        if p1[0] == p2[0] and p1[1] == p2[1]:
            return False

        length = list()
        length.append(self.distance(p1, p2))
        length.append(self.distance(p1, p3))
        length.append(self.distance(p1, p4))
        length.append(self.distance(p2, p3))
        length.append(self.distance(p2, p4))
        length.append(self.distance(p3, p4))

        length.sort()  # 6条边：四条边+2条对角线

        # 将距离排序后：前4条边都相等，后面2条对角线相等，且对角线满足勾股定理
        if length[0] == length[3] and length[4] == length[5] and length[0] * 2 == length[5]:
            return True
        else:
            return False

    def distance(self, start: List[int], end: List[int]) -> int:
        return pow(start[0] - end[0], 2) + pow(start[1] - end[1], 2)

3.有效三角形的个数

题目：

给定一个包含非负整数的数组 nums ，返回其中可以组成三角形三条边的三元组个数。

示例 1:

输入: nums = [2,2,3,4]
输出: 3
解释:有效的组合是: 
2,3,4 (使用第一个 2)
2,3,4 (使用第二个 2)
2,2,3

示例 2:

输入: nums = [4,2,3,4]
输出: 4

题解：


class Solution:
    def triangleNumber(self, nums: List[int]) -> int:
        # 2344 2234 223456789 12489
        nums.sort()
        n=len(nums)
        ans=0
        for k in range(2,n):
            i,j=0,k-1
            while i<j:
                if nums[i]+nums[j]>nums[k]:
                    ans+=(j-i)
                    j-=1
                elif nums[i]+nums[j]<=nums[k]:
                    i+=1
        return ans
