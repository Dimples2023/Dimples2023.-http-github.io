1.两数相加

描述：给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

题解：

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        c1=0
        node1=l1
        n1=0
        while(1):
            if node1!=None:
                n1=n1+node1.val*(10**c1)
                c1=c1+1
                node1=node1.next
            else:
                break
        c2=0
        node2=l2
        n2=0
        while(1):
            if node2!=None:
                n2=n2+node2.val*(10**c2)
                c2=c2+1
                node2=node2.next
            else:
                break
        sum=n1+n2
        l3=ListNode()
        node=l3
        while(1):
            val=sum%10
            node.next=ListNode(val)
            sum=sum//10
            node=node.next
            if sum==0:
                break
        return l3.next

更优解：

思路：

依次遍历L1跟L2链表相加，相当于是个位十位百位依次相加
如：
L1 = [1,4,2], L2 = [4,6,5]
可看成：241 + 564 = 805
结果为：[8,0,5]

我们将每位数相加后除以10的余数记为val（即该位的数字）
我们将每位数相加后除以10的商记为tmp（即下一位的进位）

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        # 初始化链表
        head = tree = ListNode()

        val = tmp = 0
        # 当三者有一个不为空时继续循环
        while tmp or l1 or l2:
        #注意tmp也不为0（即没有进位）时才结束，因为可能存在相加之后的数的位数大于l1和l2的情况 下图所示           
			val = tmp
            if l1:
                val = l1.val + val
                l1 = l1.next
            if l2:
                val = l2.val + val
                l2 = l2.next

            tmp = val // 10
            val = val % 10

            # 实现链表的连接
            tree.next = ListNode(val)
            tree = tree.next

        return head.next

2.无重复字符的最长子串

知识点：滑动窗口 字符串 集合 字典

描述：给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

题解：

1.运用集合
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        max_len=0
        Charset=set()
        i=0
        for j in range(len(s)):
            while s[j] in Charset:
                Charset.remove(s[i])
                i+=1
            Charset.add(s[j])
            max_len=max(max_len,j-i+1)
        return max_len
2.运用字典
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        #初始化
        max_len=0
        dic={}
        start=0
        for end in range(len(s)):
            dic[s[end]]=dic.get(s[end],0)+1
            if len(dic)==end-start+1:
                max_len=max(max_len,end-start+1)
            else:
                while end-start+1>len(dic):
                    dic[s[start]]-=1
                    if dic[s[start]]==0:
                        del dic[s[start]]
                    start+=1
        return max_len
